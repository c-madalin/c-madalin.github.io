Quizzard

AI-powered educational platform for interactive learning

Developed: March 2024
Type: Hackathon Project
Team Size: 3 Members
GitHub Repository

Project Overview

Quizzard is an AI-powered educational platform that enhances learning through interactive document-based studying and adaptive quizzes. The platform was developed during the CodeStorm Hackathon, where we aimed to create a solution that would make studying more efficient and personalized.

The Challenge

Traditional studying methods often lack interactivity and personalization, making it difficult for students to engage with material effectively. We wanted to create a platform that would allow students to interact with their study materials in a more dynamic way, while also providing teachers with tools to create customized assessments quickly.

Key Features

  • Upload and interact with study materials (PDFs, documents) via an AI chat interface
  • Generate adaptive quizzes customized to the uploaded content
  • Support for multiple question types (multiple-choice, single-choice, true/false)
  • Teachers can quickly create custom assessments
  • Integration with AI models (Ollama, Mistral, Gemma B2) for intelligent responses
  • Vector database (Chroma DB) for efficient document retrieval

Technologies Used

Flask Python Ollama Mistral Gemma B2 Chroma DB HTML/CSS JavaScript Bootstrap

Development Process

Day 1

Ideation & Planning

Brainstorming sessions to identify educational pain points and conceptualize our solution. We mapped out the core features and user flows for both students and teachers.

Day 2

Backend Development

Set up the Flask framework and integrated AI models (Ollama, Mistral, Gemma B2) for document processing and chat interactions. Implemented Chroma DB for vector-based document storage and retrieval.

Day 3

Frontend Development

Created responsive user interfaces for document upload, chat interaction, quiz generation, and teacher dashboard. Implemented real-time interactions using JavaScript and Bootstrap for styling.

Day 4

Testing & Refinement

Conducted user testing with students and teachers to identify usability issues and gather feedback. Refined the AI responses and quiz generation algorithms based on test results.

Day 5

Presentation & Demo

Prepared our pitch and demonstration for the CodeStorm Hackathon judges. Showcased the platform's capabilities with live demos of document interaction and quiz generation.
